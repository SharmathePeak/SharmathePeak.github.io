about
~$cat about.txt

I'm Vishnu Sharma, a self-taught systems programmer currently studying at BITS Hyderabad. I've been coding since before college, teaching myself C, C++, Python, and low-level systems work from scratch — because I actually want to know what's happening under the hood.

My projects lean toward the foundational: neural nets without libraries, a Brainfuck interpreter, terminal apps, 2D games in pure C++. I'm also into Linux ricing on Arch with Hyprland, drawing, and learning the path from C all the way to Assembly.
